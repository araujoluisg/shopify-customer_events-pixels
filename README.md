# Shopify-customer_events-pixels
Desenvolvimento de camada de dados para eventos de GA4 Ecommerce via customer_events Shopify

Eventos implementados:

- page_view
- view_item_list
- select_item
- view_item
- add_to_cart
- view_cart
- remove_from_cart
- begin_checkout
- add_payment_info
- add_shipping_info
- purchase
